**NOP**: No Operation
---------------------------------------

The ``NOP`` instruction does not have an effect when executed. It is usually
used as a placeholder occupying exactly one byte.

.. list-table:: Opcodes
   :header-rows: 1

   * - Instruction
     - Opcode
     - Cycles
     - Affected Flags
   * - NOP
     - 0xEA
     - 2
     - none
